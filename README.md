# Projet Fil Rouge - CYNA Infrastructure Modernization

## Présentation

Ce projet a pour objectif de concevoir et documenter la modernisation complète du système d'information de l’entreprise CYNA dans le cadre de son déménagement à Genève et de l’ouverture d’une filiale à Paris.

L’infrastructure doit répondre à des exigences élevées en matière de :

- Cybersécurité
- Haute disponibilité
- Scalabilité
- Performance réseau
- Automatisation
- Continuité d’activité

## Objectifs principaux

- Déployer une architecture Cloud Hybride
- Interconnecter Genève et Paris de manière sécurisée
- Mettre en place une segmentation réseau avancée
- Intégrer des solutions SOC / EDR / XDR / SIEM
- Déployer PRA / PCA
- Industrialiser l’infrastructure via Terraform / Ansible
- Produire une documentation technique complète

## Technologies envisagées

### Infrastructure

- Proxmox / VMware
- AWS / Azure / Cloud Hybride
- VLAN / VPN / SD-WAN
- Firewall NGFW

### Cybersécurité

- CrowdStrike
- SentinelOne
- Palo Alto Cortex XDR
- Splunk / QRadar

### Automatisation

- Terraform
- Ansible
- GitHub / CI-CD
